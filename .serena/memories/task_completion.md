# Task Completion Checklist

## Before declaring a task complete:
1. Run `npm run lint` - check for errors
2. Run `npm run build` - ensure build passes
3. Test in Firefox with `npm run preview`
4. Check git status for changed files

## Suggested commit command format (print for user):
```bash
git commit -m 'short, descriptive message of the changes - 3 lines or less'
```
